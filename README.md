# LawBridge V2

A proper Astro + Decap CMS rebuild for LawBridge.

## Local development

```bash
npm install
npm run dev
```

## Netlify settings

Build command:

```bash
npm run build
```

Publish directory:

```bash
dist
```

## CMS

Admin URL:

```text
/admin/
```

Netlify Identity and Git Gateway must be enabled.
